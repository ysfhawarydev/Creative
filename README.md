# 🎨 Creative

## 📌 Overview
Creative is a modern and fully responsive website built using HTML, CSS, and Bootstrap.  
The project focuses on delivering a clean, elegant, and professional UI design with a smooth and structured layout that works on all devices.

---

## ✨ Features
- Fully responsive design using Bootstrap  
- Clean and modern UI  
- Professional and elegant layout  
- Mobile-friendly design  
- Well-structured sections  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- Bootstrap 5  

---

## 🎯 Purpose
This project was built to practice frontend development skills and improve UI design using Bootstrap and responsive web design principles.
